Master Ledger System

📊
Client
Construction Company
Category
Google Sheets
Tools
Google Sheets, Apps Script
The challenge

The problem

The company tracked jobs across dozens of individual spreadsheets. There was no central overview of all jobs, data was duplicated and often out of sync, and finding any historical record required hunting through multiple files.

The solution

What I built

I built a multi-tab master ledger system with a central MASTER LEDGER tab and individual per-job tabs. Every row has a unique Sync ID (UUID). Checkbox-triggered scripts copy rows bidirectionally, with duplicate prevention and automated date-based sorting keeping everything organized automatically.

Features

Key features delivered

✓
UUID-based Sync ID for every row
✓
Bidirectional sync between MASTER and job tabs
✓
Checkbox-triggered row copying automation
✓
Duplicate prevention logic
✓
Automated date sorting (3-bucket system)
✓
SETUP tab for configuration
✓
Full reconciliation on demand
Result

“The team eliminated all manual copy-pasting between sheets. The master ledger is always up to date without any manual effort.”
